# RADiCAL papers — CERN SPS May 2023 dataset

Two papers are planned from this one dataset (the settled 2-paper strategy; see the memory
`radical_apparatus_conclusions.md` → ★★ PAPER PLAN, and `docs/PAPER_OUTLINES.md`). Both build
on the published parent paper **Perez-Lara, Wetzel, Akgun et al. (RADiCAL), NIM A 1068 (2024)
169737** (arXiv:2401.01747) and cover the roadmap in its §7.

- **Paper 1** thesis: *light yield, not WLS species, sets the time resolution.* Method contribution:
  the HG/LG-ratio CFD recovers the saturated edge (and removes the published 0.2 ns satellite).
  Covers published roadmap §7.2. Nearly ready.
- **Paper 2** thesis: energy and transverse position from the same shower-max corner-capillary
  signals, and a simultaneous time+energy+position ("4D") demonstration in one TENERGY module.
  Covers published roadmap §7.1, §7.3/7.4 + the E-type energy capillary. Needs the localization
  analysis built and figures finalized.

## Conventions (both papers)
- LaTeX: `elsarticle` (NIM A), two-column. Build with `tectonic <main>.tex` from inside the paper
  folder (first run fetches the class online). `\graphicspath{{figs/}}`.
- Each paper folder is self-contained: `<main>.tex`, `figs/` (PNG copies of the analysis outputs),
  any `\input` tables, plus working notes (`README.md`, `OUTLINE.md`, `THREAD.md`).
- **Figures are generated by macros under `analyze/studies/`** and land in `figures/narrative/`
  (Paper 1) or `site/assets/` (Paper 2); the per-paper `figs/` holds the copies the `.tex` includes.
  The authoritative figure index — path, macro, what it shows, conclusion, paper-status — is the
  memory `figure_catalog.md`.
- Working prose drafts and outlines live in `docs/` (`PAPER1_DRAFT.md`, `PAPER2_DRAFT.md`,
  `PAPER_OUTLINES.md`); the formal manuscripts live here.
