Research & Development of Optical Materials

Ultra-compact, fast-timing EM calorimetry

An index-finger-sized tungsten / LYSO:Ce sampling module that times electromagnetic showers to tens of picoseconds at shower maximum, read out through wavelength-shifting capillaries. A radiation-hard R&D program — and a reconfigurable test bed for optical materials.

25.7 ± 0.6 ps
brightest 150 GeV showers at shower max (≈50 ps full fiducial)
18.8 ± 0.8 ps
fitted floor — confirms the published 17.5 ps

Get up to speed

Four steps from never-heard-of-it to running the analysis and simulation yourself. Follow them in order, or jump to whichever you need.

Understand it

What RADiCAL is, and how a burst of shower light becomes a picosecond-precise timestamp — no background assumed. Then open the real test-beam data in your browser and make a plot yourself.

Set up the code & data

Clone the repository and source the environment once per shell. The analysis-ready ntuples (≈13 GB) are fetched separately; the guides below walk through it.

# clone & enter git clone github.com/jwwetzel/RADiCAL cd RADiCAL && source setup.sh
